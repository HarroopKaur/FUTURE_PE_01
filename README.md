## 📌 Project Overview

This project demonstrates a structured prompt engineering system designed to generate high-conversion website copy for local businesses.

Rather than generating random content, this system uses:

- Role-based prompting
- Business context injection
- Modular output generation
- Tone adaptation controls
- Iterative optimization

The goal is to simulate real-world client website copy development using AI tools.

## 🧠 Prompt System Architecture

The structured framework includes:

1️⃣ Role Assignment Prompt  
2️⃣ Business Context Input Template  
3️⃣ Homepage Generation Prompt  
4️⃣ Services Page Generation Prompt  
5️⃣ CTA Optimization Prompt  
6️⃣ Tone Adaptation Logic  

This modular structure ensures reusability for different clients and industries.

---

## 🔄 Iteration & Optimization Process

The workflow followed a structured improvement cycle:

1. Initial output generation (Version 1)
2. Critical evaluation of clarity, persuasion, and specificity
3. Prompt refinement to improve outcome-driven messaging
4. Optimized output generation (Version 2)
5. Final selection and polishing

This demonstrates applied LLM optimization rather than one-step content generation.
