# Prompt Framework Design
AI Website Copy Generator System

## 1. Role Assignment Prompt

"You are a professional website copywriter who specializes in writing high-conversion website copy for local service businesses. 
Your writing must be clear, benefit-driven, persuasive, and simple.
Avoid generic AI phrases. 
Write as if this is for a real paying client."

---

## 2. Business Context Input Template

---

## 3. Homepage Generation Prompt

"Using the business details provided, generate:

1. A strong homepage headline with a clear value proposition.
2. A supporting sub-headline explaining who it’s for and the key benefit.
3. A short introduction paragraph (3–4 lines).
4. A clear call-to-action section.

Make the content persuasive and website-ready."

---

## 4. Services Page Generation Prompt

"Based on the business information:

1. List the main services.
2. Write short benefit-focused descriptions for each.
3. Add a short 'Why Choose Us' section.
4. Keep language simple and professional."

---

## 5. CTA Optimization Prompt

"Write 3 different high-conversion call-to-action sections:
- Urgency-based
- Trust-based
- Location-based

Keep them short and compelling."

---

## 6. Tone Adaptation Modifier

"If tone style is:
- Friendly → Use warm, conversational language.
- Professional → Use clear, confident, structured wording.
- Confident & Modern → Use bold, forward-looking language.
"
